### Istruzioni per l'uso
per lanciare l'applicazione da CLI eseguire i seguenti comandi:

Windows CMD:
```windowscmd
> set FLASK_APP=hello
> flask run
```

Windows PowerShell:
```PowerShell
> $env:FLASK_APP = "hello"
> flask run
```

UNIX like os:
```shell
$ export FLASK_APP=hello
$ flask run
```
